Installation

To install WooCommerce Checkout Fields & Template Documentation, is not much different to installing any other (free) WordPress plugin, so you might be familiar with this process already. If not, please follow instructions below.

You can install by using WordPress uploader or using FTP

WordPress plugin uploader

  1. Log into your website
  2. From admin menu, hover to plugin then click Add new
  3. Click Upload
  4. Browse to the woocommerce-checkout-fields-templates zip file (file that you extract after download from Codecanyon).
  5. Click Install now and wait while plugin is uploaded to your server

FTP

You can also install Panpan via FTP

  1. Browse to the plugin's zip file (it is located in the folder you've downloaded from CodeCanyon) and extract that file, you will get folder named "woocommerce-checkout-fields-templates"
  2. Open your FTP program, FTP into your site and navigate to the wp-content/plugins/ directory
  3. Upload woocommerce-checkout-fields-templates folder to this directory
  4. Navigate to your Control Panel: Plugins
  5. Under Panpan, click Activate

Installing Updates

When plugin has a bug fixes or enhancements, you will be notified to upgrade.

Installing updates is just involves replacing the old plugin files with the new ones. It's always a good idea to backup everything before you upgrade, especially if you have made any changes to Panpan's files, as the upgrade will overwrite all files. Configure that you saved though WordPress admin will all remain intact.

Backing Up

  1. It is always a good idea to back up your plugin before making any changes
  2. Open your FTP program and FTP into your site and navigate to the wp-content/plugins/ directory.
  3. Download the woocommerce-checkout-fields-templates folder and save it as a backup

Manual update

  1. Download the new version of the plugin from CodeCanyon
  2. Browse to the plugin's zip file (it is located in the folder you've downloaded from CodeCanyon) and extract that file, you will get folder named woocommerce-checkout-fields-templates
  3. Switch to your FTP program, FTP into your site and navigate to the wp-content/plugins/ directory.
  4. Transfer the new woocommerce-checkout-fields-templates folder into the wp-content/plugins/ directory on your server, overwrite the existing one.
  5. Ensure that your permissions are set properly on the plugins/woocommerce-checkout-fields-templates directory. If your server can't read the files, you might run into errors. This can happen depending on how your FTP account is set up.

Note: If you are running a local server, just copy and override the files.

Row settings

Row options

Row will be the wrapper for entire your column & fields, on each row, you can add multiple fields and so on.
Row distinction: You will know it's row by dashed blue border

Available options:

  1. Move row: click on icon to move the position of current row.
  2. Column devider: click on icon to select the number of column for your row, if the pre-made column doesnt fit to your need, click on to input the custom column value.
  3. Row edit: click on icon to open the setting for row, you can choose the custom ID, class, row type.
  4. Row styling: You can input the padding, margin or border by px or % for row by click on icon .
  5. Add row: Click on icon to add new column quickly
  6. Remove row: Click on icon to remove current column.

Element settings

Elements - Global Options

By default, global options has 2 tab, General options & Advanced options
  1. General options: This is the options appear across all the fields, they're: Title,Placeholder, Field ID, require (checkbox), Display field on email (checkbox) and Display field on order detail (checkbox).
  2. Advanced options: On this tab, you can add or edit the class of the fields, you're also can enable the conditional for that field, which will make field show/hide on specific conditions.
    1. Extend With custom fields, there's pricing option, you can add an amount of money for that field, that mean user will be charged fees if they choose, type in, or select that field.

Elements - Specific Options

  1. Title: title tag is the option which you can choose from h1 to h6.
  2. Radio: single selection. The values should be put in as this pair: value(spacebar):(spacebar)label, separate between pairs by enter.
    1. Extend You can make label show up with images instead of text. To do that, check image box, and put in value pair as: value(spacebar):(spacebar)url_to_image. Exp: hi : http://demo.comfythemes.com/woocommerce-checkout-template/wp-content/uploads/2015/07/card-01.png
  3. Checkbox: The multiple selection. The values should be put in as this pair: value(spacebar):(spacebar)label, separate between pairs by enter. Exp: hi : Hello World
    1. Extend You can make label show up with images instead of text. To do that, check image box, and put in value pair as: value(spacebar):(spacebar)url_to_image. Exp: hi : http://demo.comfythemes.com/woocommerce-checkout-template/wp-content/uploads/2015/07/card-01.png
  4. Dropdown: The multiple & single selection. The values should be put in as this pair: value(spacebar):(spacebar)label, separate between pairs by enter. Exp: hi : Hello World
    1. Extend You can enable multiple select by check box Multiple select.
  5. Date picker: will enable user to choose the date on the calendar.
    1. Style We have created several style for the calendar, you can choose from the style dropdown, the prebuilt styles include Cangas, Latoja, Lugo, Melon, Nigran, Santiago, Siena, Vigo.
    2. Min - Max date By select the min date, max date, your customer only can choose the the date range you choose. Exp: min: 01/18/2015, max: 01/22/2015, so your user only can choose the date from the date 18 to 22
    3. Except weekend enable if you dont want to deliver your product on weekends date (Saturday and Sunday).
    4. Specific dates enable if you only deliver your products on specific dates.
  6. Text field: Single line text input field.
  7. Text area: Multiple line text input field, width and height extandable.
  8. Email: Validate the input with email type.
  9. Phone: Validate the input with number type.
  10. File upload: Allow user to upload file via checkout page, the selected file only uploads when order is created, no matter the order status is success or pending.
      1. Note: For your secure, we made the upload field check the files uploaded with header checking, and it require PHP Extension: fileinfo. Please check with your hosting provider if it doesnt load, your upload form wont work.

How to build template

We will guide you how to build template from the ground.

Please note the following detail: Class (most important):
  • Billing column must be filled with class ossvn-billing-details
  • Shipping column must be filled with class woocommerce-shipping-fields
  • Order review and Payment Method must be filled with class order_review
